HandiPressante

Collaborative mobile application to geolocalize disabled toilets

What is HandiPressante ?

Application that helps finding disabled toilets for everyone

Although primarily intended for persons with disabilities, HandiPressante can be useful to anyone in search of toilet near his position.

Collaborative application

HandiPressante offers users to add and edit content in a simple and transparent way.

An application adapted to users

HandiPressante provides a settings menu to adapt the user interface for individual needs.

Main features of the application

Search by list & map

The application allows you to find the nearest disabled toilets from your current position found by ​​geolocation. Moreover, a map give you the possibility to look for disabled toilets at a desired location.

Add comments

Users can give their opinion based on their own experience to help the community. They can also evaluate various criteria such as cleanliness, accessibility...

Change the user interface

In order to accommodate to many people, settings of the user interface can be changed easily with a dedicated menu to enhance application usage : button size, navigation modes...

Add unreferenced toilet

Users can add missing toilets to the existing list, and add information about these toilets (photos of the building, description...)

More information

This application is powered by the computer science department of INSA Rennes in partnership with the Réseau Breizh Paralysie Cérébrale. This project aims to create an application for people with disabilities to improve their daily lives. For certain diseases, the presence of toilets near their position is a necessity. Therefore, the application attempts to provide them with a simple and effective solution. The community aspect of the application allows users to get involved and to extend the scope of the application.
